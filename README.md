# What_is_Combineino? コンバインーノとは？
Arduino互換MPUといろんなシールドのコンバイン（結合）したい人のオープンハードウェア

スニペットとは、一般的には「切れ端」「断片」という意味の英語である。 IT用語としては、プログラミング言語の中で簡単に切り貼りして再利用できる部分のこと、または、検索エンジンによる検索結果の一部として表示される、Webページの要約文のことである。
